# GIS Test Task — ArcGIS API for Python

Тестове завдання на позицію **GIS Python Developer**.

Скрипт реалізує:
- обробку табличних даних,
- трансформацію значень у відповідності до умов завдання,
- завантаження результатів у **Hosted Feature Layer** в ArcGIS Online
  з використанням **ArcGIS API for Python**.

---

## Опис задачі
Вихідні дані представлені у вигляді таблиці з датами, регіонами, містами,
набором числових показників (`Значення 1 – Значення 10`) та координатами точок.

Завдання включає два основних етапи:
1. **Обробка таблиці**  
   - кожен рядок трансформується у кілька рядків відповідно до значень
     у колонках `Значення 1 – Значення 10`;
   - значення перетворюються у бінарний формат (0 / 1);
   - рядки, де всі значення дорівнюють 0, зберігаються як один рядок.

2. **Завантаження у ArcGIS Online**  
   - підготовлені дані завантажуються у існуючий Hosted Feature Layer;
   - геометрія точок створюється на основі координат `long / lat`;
   - атрибутивні поля мапляться до схеми шару;
   - демонструється додавання та оновлення об’єктів через API.

## Технології
- Python 3.x
- pandas
- ArcGIS API for Python
- python-dotenv

---

## Структура проєкту
- `data/input.csv` — вхідні дані (приклад)
- `src/transform.py` — трансформація таблиці (розмноження рядків)
- `src/arcgis_upload.py` — робота з ArcGIS Online (add/update)
- `src/main.py` — точка входу (читання CSV → transform → upload)

## Запуск проєкту

1. Створити та активувати віртуальне середовище
2. Встановити залежності
3. Створити файл .env у корені проєкту:

```env
ARCGIS_URL=https://www.arcgis.com
ARCGIS_USERNAME=your_username
ARCGIS_PASSWORD=your_password
ARCGIS_ITEM_ID=c51cb411da1a468aaed719085337cc2b
```

4. Запустити скрипт:

```
python src/main.py
```

## Реалізовані можливості ArcGIS API

У межах завдання використано:
- arcgis.gis - авторизація та доступ до контенту порталу;
- arcgis.features - робота з Feature Layer;
- Accessing Item Resources - отримання властивостей та ресурсів Item;
- Appending Features - додавання нових об’єктів у шар;
- Updating Features - оновлення атрибутів існуючих об’єктів;
- Retrieve item and layer properties - отримання метаданих шару.
